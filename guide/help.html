Installation:

Go to Github and install the following:

https://github.com/ArtGames101/Aroba

Then go to https://python.org/ and install Python3 or Python3.5

When installed open the ArtSystem Folder in a command prompt or terminal and run “python3 run.py”

Done!





Login File Error:



This error means your Username, password or avatar has stopped working! If you want to fix this there is 3 options Register, Guest Account or Restore!



Register:

With the register option you can create a new account so that you have new data for ArtSystem to read! It will fix the File Error!

Guest Account:

Temporarily Lets you use the guest account until you register



Restore:

The Restore option is currently in BETA stage but it works! The Restore option will restore Username and password!
